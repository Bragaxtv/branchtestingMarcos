## Usar como extensão

Este repositório pode ser adicionado como **extensão** no MakeCode.

* abrir [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* clique em **Novo Projeto**
* clique em **Extensões** em baixo do menu com ícone de engrenagem
* procure por **https://github.com/lucassouzacodigos/branchtesting** e importe

## Editar este projeto

Para editar este repositório no MakeCode.

* abrir [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* clique em **Importar** e depois clique em **Importar URL**
* cole **https://github.com/lucassouzacodigos/branchtesting** e clique em importar
